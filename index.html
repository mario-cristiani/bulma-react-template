Hello. You can use this email creation tool to help promote your project.

While we can help you apply the VIT EAS style and get it loaded into our project repository, we can't write the email for you. So, before you get started, take some time to think about what the project means for the end user community and build a short story that is relevant.

Here are some questions you can try to answer:

  • Is this project part of a larger program of strategic initiative.
  • What was the current process that was being used? Is this brand new or an enhancement?
  • Are there follow on projects that will continue

To get started watch the short videos below. In a nut shell, you need to: assemble the content, input it into the email generator, import the email template and images into an outlook email, test and send.
