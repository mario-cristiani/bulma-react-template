Welcome, to the “EAS Project Email Generator”. We hope it helps you communicate your project story to the business community. To jump start, follow the path below.

#1 Take a few minutes to think about what the project means in the eyes of the business and our users. Then create a simple story with the “Five Ws (Who, What, When, Where, Why) and How” in mind. Here are some hints:

  1. Is this project part of a larger program of strategic initiative?
  2. What are the pain points to be resolved?
  3. What is next?


#2 Generating the template is easy.

Just follow the instructions below each input field. Note that the Details, Key Business Capabilities, and KPI/Benefits fields can interpret markdown so that you can style the text you are importing.

When ready click the Get Some button. While there, you can download a zip file that contains all the template images.[Insert Graphic Here]

#3 If this is your first time inserting HTML into an email, you will probably need to add a button to the quick access bar in Outlook. [Insert images with more instructions here]

Open a new email and click the Attach File Button. Navigate to the HTML file and then make sure to click the dropdown arrow and select Insert as Text option. [Insert images here]

To insert images click on the text in the header, title area and footer one at a time. Then click the insert menu and Pictures. Choose the image, with the same name as the test you highlighted. [Insert images here]

Why are we doing this? By inserting the iamges manually, we ensure that people will not get an empty boxes where pictures were not downloaded from a server.

With the HTML and images incorporated, you are almost done. Double check your message formatting. Most likely you can improve the symmetry of the message with a few empty rows. As well, make sure to delete the [optional text] elements if they were unused.
